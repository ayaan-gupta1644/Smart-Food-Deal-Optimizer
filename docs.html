C Backend API Reference
HTTP/1.1 server on localhost:8080 · All responses are JSON · CORS enabled
GET /api/dishes
Lists all dishes in the database. Internally iterates the HashMap key set.
🟠 DSA: HashMap — all inserted keys enumerated · O(n)
// Sample response { "dishes": ["Butter Chicken", "Pizza", "Biryani", ...], "count": 15 }
GET /api/search?q={prefix}
Returns all dish names matching the given prefix. Powered by the Trie data structure implemented in C.
ParamTypeDescription
qstringPrefix to search (e.g. "but", "bi")
🔵 DSA: Trie — traverses prefix in O(m), DFS collects results · O(m + k)
// GET /api/search?q=bi { "algo": "Trie", "complexity": "O(m) where m=2", "prefix": "bi", "count": 1, "results": ["Biryani"] }
GET /api/prices?dish={name}
Fetches prices for a dish across all 3 platforms. Uses HashMap for O(1) lookup by dish name.
ParamTypeDescription
dishstringExact dish name (case-insensitive)
🟠 DSA: HashMap — djb2 hash → bucket index → chain traversal · O(1) avg
// GET /api/prices?dish=Biryani { "algo": "HashMap", "complexity": "O(1)", "hash_bucket": 42, "dish": "Biryani", "platforms": [ { "platform": "Swiggy", "base": 220, "delivery": 25, "tax": 11, "total": 256 }, ... ] }
GET /api/coupons?dish={name}&platform={platform}
Returns all coupons applicable for a dish on a specific platform, with savings pre-calculated.
ParamTypeDescription
dishstringDish name
platformstringSwiggy | Zomato | Magicpin
🟢 DSA: Greedy pre-analysis — filters by platform + min_order · O(n)
GET /api/optimize?dish={name}
The main endpoint — runs all 5 DSA algorithms and returns optimal deal with full algorithm trace.
🟠 HashMap O(1) → 🔵 Trie O(m) → 🟢 Greedy O(n) → 🟡 DP Knapsack O(n×W) → 🟣 Min-Heap O(n log n)
// GET /api/optimize?dish=Pizza { "dish": "Pizza", "algo_trace": { "hashmap": { "complexity": "O(1)", "bucket": 77 }, "trie": { "complexity": "O(m)" }, "greedy": { "complexity": "O(n)", "log": [...] }, "dp_knapsack":{ "complexity": "O(n*W) n=6 W=200", "log": [...] }, "min_heap": { "complexity": "O(n log n)", "size": 3 } }, "rankings": [ { "rank": 1, "platform": "Zomato", "final_price": 245, "saving": 100, "coupons_used": "FIRST100" }, ... ] }
